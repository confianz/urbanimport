Odoo Braintree Payment Gateway

Braintree is a division of PayPal and full-stack payments platform that makes it easy to accept payments in your app or website. It's service replaces the traditional model of sourcing a payment gateway and merchant account from different providers. From single-touch payments to mobile SDKs and foreign currency acceptance, they provide everything you need to start accepting payments today. With this module, you will be able to accept payment from Odoo directly in your braintree account in just minutes!!

Key Features

Latest JS SDK

Built on latest Braintree JavaScript v3 SDK (Software Development Kit)

Drop-in UI

Integrated Drop-in UI that offers a quick and easy way to securely accept payments

Support Multiple Website

Works smoothly in Multi Website feature of Odoo!!

Multi Merchant & Currencies

Added support to handle multiple merchant accounts having different currencies.

Manage Credit/Debit Cards

Process credit card, debit card transactions from the Odoo e-Commerce Website.

Accept CVV

Can accept payment with security code (CVV) based on configuration of your Braintree Account

Saved Cards

Checkout process experience made easy using saved cards!!

Server to Server Process

No redirection to Braintree website. Easy payment completion from Odoo it self via s2s method.

Odoo CE & EE

Compatible with both Odoo Community (CE) and Enterprise (EE)

Secured Transaction

Secure and user friendly Interface that helps people to checkout easily on your website..!!
